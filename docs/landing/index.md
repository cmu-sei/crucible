# Core Application Documentation

We provide comprehensive user and administrator guides for each core application, covering setup, configuration, and day-to-day use within the Crucible Framework.

- *[Alloy Guide](../alloy/index.md)*: Use Alloy for on-demand event launching and exercise management with reporting.

- *[Blueprint Guide](../blueprint/index.md)*: Blueprint supports collaborative Master Scenario Event List (MSEL) creation.

- *[Caster Guide](../caster/index.md)*: Caster enables infrastructure-as-code design and deployment using OpenTofu.

- *[CITE Guide](../cite/index.md)*: CITE supports multi-organization cyber incident evaluation and scoring.

- *[Gallery Guide](../gallery/index.md)*: Gallery provides tools for cyber incident data review and analysis.

- *[Gameboard Guide](../gameboard/index.md)*: Gameboard supports cybersecurity game design and competition management.

- *[Player Guide](../player/index.md)*: Player provides a user interface for accessing virtual environments and managing team membership.

- *[Steamfitter Guide](../steamfitter/index.md)*: Steamfitter handles scenario task organization and execution via StackStorm.

- *[TopoMojo Guide](../topomojo/index.md)*: Use TopoMojo for form-based lab and challenge creation with virtual machine configuration.
